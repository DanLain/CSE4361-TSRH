Patient Profile Creation!

This page displays fields to be entered for a new Patient record.

  • Patient Details

    Guidelines to create a new Patient record.

    • To add a new patient all the fields must be entered.
    • TSRH ID format should be 6 digits, it does not take greater than or less than six digits or any other special characters. Example:123456
    • First Name, Last Name and Mother's Name cannot contain numbers or special characters.
    • Enter the country code. Example: For United States +1
    • Phone number should be 10 digits. Example: 8172722201 or 817-272-2201
    • Image should be either of the following formats only jpg|jpeg|bmp|png.
